# New Client Homepage Demo Design

## 背景

`newclient/h5demo` 当前用于新客户端设计阶段，不进入真实开发。

为了避免在首页 demo 阶段盲目扩能力，本设计严格以前置盘点文档《现有 client 接口与功能盘点》为基础，只设计当前 `client` 已经具备真实承接能力的首页形态。

## 设计目标

本轮只产出首页 demo，且 PC 与 H5 分开设计：

- PC 首页：更像中文金融站 / 券商研究终端首页
- H5 首页：更像国内金融 App 首页
- 两端共享同一套蓝金金融视觉系统
- 首页核心主线固定为：
  - 股票 / 期货推荐
  - 研报 / 市场资讯

## 已确认方向

### PC 首页

采用“双塔首页”结构：

- 左侧主工作区承接推荐主线
- 右侧承接焦点研报与市场导读
- 推荐比研报更重，但研报必须在首屏有独立存在感

PC 首页核心模块固定为：

1. 顶部站点导航
2. 推荐主 Hero
3. 为什么选择该推荐
4. 历史推荐样本预览
5. 焦点研报卡
6. 市场资讯导流
7. 底部信息区

### H5 首页

采用“A1”方向，明确向国内金融 App 首页收敛：

- 顶部品牌头与搜索 / 消息入口
- 快捷入口宫格前置
- 轻量数据条
- 推荐主卡作为第一主视觉
- 焦点研报大卡作为第二主线
- 历史样本卡与资讯列表顺序下沉
- 底部 Tab 固定存在

H5 首页核心模块固定为：

1. 顶部品牌区
2. 主线 Hero
3. 快捷入口宫格
4. 轻量数据条
5. 推荐主卡
6. 推荐理由 / 风控说明
7. 焦点研报卡
8. 历史样本预览
9. 市场资讯列表
10. 精简底部信息
11. 底部 Tab

## 内容边界

本次首页 demo 只能使用当前真实能力可承接的内容，不新增以下能力：

- 大盘指数横排数据流
- 热门题材榜
- 用户社区 / 发帖 / 聊天室
- 直播、短视频
- 开放平台
- 自建云端自选股

允许以首页静态文案方式展示的真实能力包括：

- 股票推荐主位
- 期货机会承接
- 推荐理由摘要
- 风险边界与止盈止损说明
- 历史推荐样本预览
- 焦点研报与解读
- 市场资讯导流
- 会员与资讯页承接入口

## 视觉原则

- 统一蓝金金融系，不使用暖绿 / 米色主题
- PC 更像中文金融站，不像欧美 SaaS Landing
- H5 更像国内金融 App，不像简化版网页
- 文字语言偏中文金融站常见表达，不使用夸张营销语气
- 卡片信息密度适中，强调“先看结论，再看理由，再继续阅读”

## 交付物

本轮交付以下 demo 文件：

- `newclient/h5demo/home-pc-demo.html`
- `newclient/h5demo/home-h5-demo.html`
- `newclient/h5demo/home-demo-index.html`

其中：

- `home-pc-demo.html` 用于单独审查 PC 首页
- `home-h5-demo.html` 用于单独审查 H5 首页
- `home-demo-index.html` 用于统一入口与切换说明

## 审查重点

审查首页 demo 时主要看：

1. PC 是否已经像中文金融站首页，而不是海外 SaaS 页面
2. H5 是否已经像国内金融 App 首页，而不是 H5 宣传页
3. 推荐和研报是否都在首页首屏建立存在感
4. 首页是否仍然严格基于当前真实能力
5. 底部信息是否补齐，不再像半截页面
