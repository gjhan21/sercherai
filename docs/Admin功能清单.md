# Admin 功能清单（开发版）

## 1. 账号与权限
1. 管理员登录/退出
2. 角色管理：`SUPER_ADMIN`、`STRATEGY_ADMIN`、`CONTENT_REVIEWER`、`KYC_REVIEWER`、`OPS`
3. 权限管理：菜单权限、接口权限、数据权限
4. 操作审计日志：按人/时间/模块查询

## 2. 用户与实名认证
1. 用户列表：搜索、筛选、查看详情
2. 用户状态管理：启用/禁用
3. 实名审核：通过/驳回、驳回原因
4. 实名复核与申诉处理

## 3. 股票与期货内容管理
1. 股票推荐管理：创建、编辑、审核、发布、下线
2. 期货策略管理：创建、编辑、审核、发布、下线
3. 套利策略管理：跨期/跨品种/期现
4. 风险提示模板管理
5. 操作指导模板管理：方向、仓位、止盈止损、失效条件
6. 套利参数配置：历史窗口、分位阈值、Z-score 阈值、半衰期阈值

## 4. 新闻资讯管理
1. 分类管理：新增、编辑、排序、上下线、可见范围（`PUBLIC|VIP`）
2. 新闻管理：新增、编辑、提交审核、发布、下线
3. 附件管理：上传、删除、下载、大小/类型校验
4. 新闻审核流：待审、通过、驳回
5. 附件下载安全策略：签名链接有效期、防盗链、下载审计

## 5. 增长激励管理
1. 分享活动配置：活动时间、奖励规则、封顶额度
2. 邀请记录查询：邀请人、被邀请人、状态、风险标记
3. 奖励记录管理：待发放/已发放/冻结/驳回
4. 奖励审核：发放、冻结、驳回、原因记录
5. 反作弊处理：同设备/同实名/同支付账号拦截
6. 风控规则配置：阈值管理、灰度生效、规则回滚
7. 风险命中复核：放行/确认、申诉处理

## 6. 会员与订单
1. 会员产品管理：价格、时长、状态
2. 订单查询：充值记录、会员订单、支付状态
3. 异常订单处理：失败重试、人工修正
4. VIP 配额配置：阅读文档数量上限、订阅资讯数量上限
5. VIP 配额周期：按日/按月重置策略
6. 用户配额调整：单用户补偿或扣减配额
7. 奖励账户与提现审核：提现审批、打款状态、失败回滚

## 7. 支付与对账
1. 支付回调日志：验签结果、幂等处理状态
2. 日对账批次：差异订单清单、重试与补单
3. 对账告警：差异阈值告警与处理闭环

## 8. 数据源与配置
1. 数据源管理：供应商配置、状态监控
2. 数据授权与调用审计
3. 系统配置：字典、枚举、阈值参数

## 9. 运营与报表
1. 推荐效果报表：曝光、点击、转化
2. 新闻报表：阅读、下载、VIP 转化
3. 增长报表：分享 UV、注册、实名、首充、奖励成本
4. 导出报表：按时间范围导出 CSV

## 10. 建议优先级（开发顺序）
1. P0：登录权限、用户实名审核、推荐/策略审核发布、新闻管理、审计日志
2. P1：增长激励管理、会员订单管理、支付对账、运营报表
3. P2：高级风控、自动化运营规则、多维数据看板
