# 策略指标清单（v0.1）

本文列出股票与国内期货策略常用指标，用于推荐评分、风险分级、理由生成与审核参考。

**1. 股票策略指标**

**1.1 技术面指标**
1. 趋势强度 `MA` 组合斜率
2. 动量 `MACD`、`RSI`
3. 波动 `ATR`、历史波动率
4. 成交量异动 量比、换手率
5. 价量背离价差

**1.2 基本面指标**
1. 估值 `PE`、`PB`
2. 盈利能力 `ROE`、`ROA`
3. 成长性 营收同比、净利同比
4. 现金流 经营现金流同比
5. 负债水平 资产负债率

**1.3 资金与筹码**
1. 主力资金净流入
2. 大单成交占比
3. 北向资金持仓变化
4. 筹码集中度

**1.4 情绪与事件**
1. 新闻情绪分
2. 研报热度
3. 公告事件影响级别
4. 行业景气度

**1.5 风险指标**
1. 最大回撤
2. 下行波动率
3. 连续下跌天数
4. 流动性评分

**2. 期货策略指标（国内）**

**2.1 方向性策略**
1. 趋势强度 `MA` 斜率
2. 动量 `MACD`、`RSI`
3. 波动 `ATR`、历史波动率
4. 持仓量变化
5. 成交量变化

**2.2 套利策略**
1. 价差 `Spread`
2. 价差历史分位
3. 基差 `Basis`
4. 基差历史分位
5. 回归速度 `Half-life`
6. 协整检验结果

**2.3 期限结构**
1. Contango/Backwardation
2. 期限结构斜率

**2.4 风控指标**
1. 保证金占用
2. 滑点预估
3. 流动性评分
4. 主力合约切换风险

**3. 推荐评分与解释映射（建议）**
- 技术面 30%
- 基本面 30%
- 资金面 20%
- 情绪面 10%
- 风险因子 10%

**4. 策略理由生成规则（建议）**
- 仅输出 Top3 贡献因子
- 负向因子必须提示
- 风险项优先展示

**5. 策略失效判定（建议）**
1. 评分连续 N 次下跌
2. 风险等级上升
3. 触发止损阈值
4. 关键因子反转
