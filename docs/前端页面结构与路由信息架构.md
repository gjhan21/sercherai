# 前端页面结构与路由信息架构（v0.1）

本文件描述 PC Web 页面结构、路由与主要模块，作为前端实现参考。前端技术栈为 Vue 3 + Vite + Element Plus。

**1. 公共布局**
- 顶部导航：Logo、主导航、搜索、用户入口
- 侧边栏：推荐、策略、动态、新闻、持仓、会员
- 主体区域：列表 + 详情

**2. 路由结构**
1. `/` 首页
2. `/stocks` 股票推荐列表
3. `/stocks/:id` 股票推荐详情
4. `/futures` 期货策略列表
5. `/futures/:id` 期货策略详情
6. `/arbitrage` 套利列表
7. `/arbitrage/:id` 套利详情
8. `/market` 市场动态列表
9. `/market/:id` 市场动态详情
10. `/news` 新闻资讯列表
11. `/news/:id` 新闻资讯详情
12. `/public-holdings` 公开持仓/持股
13. `/messages` 站内消息
14. `/profile` 个人中心
15. `/profile/kyc` 实名认证
16. `/membership` 会员中心
17. `/orders` 订单记录
18. `/history` 浏览历史
19. `/wallet/recharge-records` 充值记录
20. `/invite` 分享推广
21. `/invite/rewards` 分享奖励
22. `/wallet/rewards` 奖励账户

**3. 页面结构**

**3.1 首页**
- 今日推荐卡片（股票/期货）
- 市场动态摘要
- 热门策略与风险提示

**3.2 股票推荐列表**
- 筛选区：风险等级、行业、评分区间
- 排序区：评分、时间
- 列表区：推荐卡片

**3.3 股票推荐详情**
- 评分总览
- 评分分项
- 风险提示
- K 线与收益曲线

**3.4 期货策略列表**
- 筛选区：方向、风险等级
- 列表区：策略卡片

**3.5 期货策略详情**
- 策略摘要
- 风险等级与仓位建议
- 合约行情图

**3.6 套利列表与详情**
- 套利类型筛选
- 价差分位与点位建议

**3.7 市场动态**
- 动态列表
- 动态详情包含触发条件与来源

**3.8 公开持仓/持股**
- 机构持股列表
- 期货持仓结构列表

**3.9 新闻资讯**
- 分类筛选（全部、公开、VIP）
- 新闻列表
- 新闻详情（正文 + 附件下载）

**3.10 消息**
- 消息列表
- 消息详情

**3.11 个人中心**
- 基础资料
- 实名状态
- 会员状态
- 会员配额状态（阅读文档剩余、订阅资讯剩余）

**3.12 浏览历史**
- 按内容类型筛选（股票/期货/新闻）
- 最近浏览列表
- 单条删除与全部清空

**3.13 充值记录**
- 充值订单列表
- 状态筛选（处理中/成功/失败）
- 时间筛选与导出

**3.14 分享推广**
- 专属分享链接
- 邀请记录
- 渠道转化统计

**3.15 分享奖励**
- 奖励规则说明
- 奖励明细列表
- 奖励状态（待发放/已发放/冻结/驳回）

**3.16 会员中心**
- 会员等级与到期时间
- 阅读文档配额：上限/已用/剩余
- 订阅资讯配额：上限/已用/剩余
- 配额重置周期展示

**3.17 奖励账户**
- 现金奖励余额与冻结金额
- 奖励流水列表
- 提现申请入口与状态跟踪

**4. 路由权限**
- `/stocks/:id` 未实名显示脱敏
- `/futures/:id` 未实名显示脱敏
- `/news/:id` 非 VIP 用户不可访问 `VIP` 新闻
- `/membership` 未实名显示实名引导
- `/invite/rewards` 仅实名用户可见
- `/membership` 配额超限时禁用对应操作并展示提示
- `/wallet/rewards` 未实名不可提现
