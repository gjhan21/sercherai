# 技术栈（简化版）

基于可维护性优先的选型，统一技术栈如下。

**1. 前端（PC Web）**
- `Vue 3 + Vite`
- UI：`Element Plus`
- 状态管理：`Pinia`
- 图表：`ECharts`
- 站内消息：`SSE`（优先）或 `WebSocket`

**2. 管理后台（Admin）**
- `Vue 3 + Vite`
- UI：`Element Plus`
- 状态管理：`Pinia`
- 权限：`RBAC`
- 审计：操作日志

**3. 服务端**
- 语言：`Go`
- 框架：`Gin`
- API 文档：`Swagger/OpenAPI`
- 任务调度：`cron` + `Redis` 简单队列

**4. 数据层**
- `MySQL`（主库）
- `Redis`（缓存/消息/限流）

**5. 运维（轻量）**
- `Docker Compose`
- 日志：基础文件日志或 `Loki`（可选）
- 监控：`Prometheus + Grafana`（可选）
