# 专题D：回测实验与自动化运营联动

最后更新: 2026-03-23  
状态: Planned

## 1. 专题目标

把当前已有的评估摘要、增长实验和运营节奏能力，升级为“研究效果可衡量、实验结果可归因、运营动作可联动”的统一平台。

## 2. 当前基线

- 研究 run 已有 evaluation meta、publish、version-history 与部分增长实验埋点
- 后端已有 experiment analytics、归因和 order dedupe 的基础能力
- Admin 已有 review、jobs、workflow、messages 等分散运营入口
- 当前缺的是统一的研究效果仓库、回测运行模型和联动看板

## 3. 专题完成标准

- run 可关联回测配置、基准、样本窗口、交易成本假设与完整报告
- 研究效果可以按 run group / version group 聚合
- 实验平台支持 cohort、journey、模板表现对比与订单级唯一归因
- Admin 能看到“研究效果 -> 用户承接 -> 会员转化 -> 留存反馈”的联动链路

## 4. 非目标

- 自动化广告投放平台
- 通用 BI 系统替代品
- 任意页面任意可编排工作流引擎

## 5. 推荐第一轮范围

1. 回测配置与运行模型
2. 研究效果仓库
3. 实验平台 v2 与订单去重统一收口
4. 运营联动看板第一版
5. 面向研究平台的自动化动作接口
