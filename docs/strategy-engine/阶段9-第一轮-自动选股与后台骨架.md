第一轮历史目标是：

- 让股票任务在 `AUTO` 模式下不再依赖人工 `seed_symbols`
- 在 Admin 中新增 `智能选股` 一级菜单
- 先落四个核心页面：
  - 总览
  - 运行中心
  - 策略配置
  - 候选与组合
- 建立 profile / run / review / publish 的最小闭环
- 保持旧客户端消费契约与旧后台入口兼容
